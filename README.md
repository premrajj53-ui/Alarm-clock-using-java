# ⏰ Alarm Clock Using Java

A simple multithreaded Alarm Clock application built using Java.  
This project allows users to set an alarm time and plays an audio sound when the time is reached.

---

## 🚀 Features

- ⏱ Set alarm using `HH:mm:ss` format
- 🧵 Uses multithreading (`Runnable` & `Thread`)
- 🔔 Plays alarm sound using Java Sound API
- ⏳ Displays real-time clock (updates every second)
- ⌨️ Stop alarm by pressing Enter
- ❌ Handles invalid input with exception handling

---

## 🛠️ Technologies Used

- Java (Core Java)
- Java Time API (`LocalTime`, `DateTimeFormatter`)
- Multithreading
- Java Sound API (`Clip`, `AudioInputStream`)
